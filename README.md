# Library OOP

A simple console-based library management system developed in Java to practice Object-Oriented Programming concepts.

## About the Project

This project was created to reinforce fundamental Java and OOP concepts, such as:

- Classes and objects
- Encapsulation
- Relationships between classes
- Lists and object management
- Basic MVC organization
- Console interaction with the user
- Dependency injection between controllers

The system allows basic management of a library, including authors, books, and loans.

## Features

- Register authors
- Register books
- Associate books with authors
- Register book loans
- Search and display books, authors, and loans
- Remove records from the system
- Manage data during program execution
